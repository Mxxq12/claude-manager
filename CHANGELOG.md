# 更新日志

## [2.1.0] - 2026-04-08

### iOS 语音输入重构 + 侧边栏拖拽修复

- iOS 语音输入改为「先录制后识别」：用 `AVAudioRecorder` 录到本地 m4a，停止后用 `SFSpeechURLRecognitionRequest` 一次性转写，彻底解决长语音中途清空/中断问题
- 语音录制改为微信式全屏浮层：麦克风图标 + 音频波形脉冲 + 时长计时 + 上滑取消
- 语音转文字预览页重做：19pt 大字体、行距 6、自动纠错、字符计数、清空按钮、自动聚焦、large detent、顶部取消/标题/发送条
- 识别中加 loading 浮层提示
- 侧边栏拖拽 overlay 卡死修复：拖文件离开窗口或落在终端区域时，sidebar 收不到 dragLeave/drop，counter 永远归不了零，overlay 永久残留。改为在 window 级别监听 `dragend`/`drop`/`mouseleave` 做兜底清理

## [2.0.0] - 2026-04-03

### Web 远程控制 + iOS 原生客户端

- 新增 Web 远程控制服务（端口 4000），支持手机浏览器远程管理 Claude 会话
- REST API：登录认证（JWT）、会话 CRUD、历史项目列表
- WebSocket 实时推送：终端输出、状态变更、自动审批同步
- Web 前端：深色主题、PWA 支持、下拉刷新、左滑删除、语音输入、终端搜索
- iOS 原生客户端（SwiftUI + SwiftTerm）：原生终端渲染、流畅滚动、捏合缩放
- 支持 Cloudflare Tunnel 公网访问
- 密码认证 + JWT 持久化（重启不掉线）
- 会话 buffer 磁盘缓存（重启后恢复终端内容）
- Hook server 固定端口（45819），重启后 hook 不报错
- 桌面端与远程端自动审批状态双向同步
- Dock badge 通知替代系统弹窗

- 会话卡片按钮布局优化、托管模式蓝色边框标记

## [1.4.0] - 2026-03-29

### 托管模式

- 新增托管模式：会话卡片上点"托管"开启，自动在项目 .managed 目录创建控制者会话
- 控制者（Opus）先与用户头脑风暴确认方案，输出 [START_EXECUTION] 后自动切 Sonnet 进入执行循环
- 执行者完成后自动转发结果给控制者审查，控制者继续指挥或标记 [TASK_COMPLETE] 完成
- 支持暂停/恢复自动执行（暂停时切回 Opus，恢复时切回 Sonnet）
- 支持随时关闭托管，自动清理控制者会话
- 内置控制者 skill，自动加载角色规则

### 会话启动优化

- 所有会话默认使用 Opus 模型启动
- 直接 spawn claude 作为 PTY 进程，不再通过 shell 启动，消除历史命令残留
- 控制者会话以 hidden 模式创建，不在侧边栏显示
- .managed 目录的会话不会被持久化和恢复

## [1.2.0] - 2026-03-26

### 会话管理与自动化增强

- 会话列表改为固定排序模式，支持拖拽调整顺序（基于 cwd 持久化，重启后保持）
- 默认主题改为 Ayu Dark（已有用户不受影响）

- 自动模式增加废话提问检测（中英文问句模式匹配），自动回复"继续，按你的方案执行"
- 新增 Rate Limit 自动恢复：检测到限流提示后解析重置时间，到时间自动发送 /resume（所有会话默认生效）
- Shift+Enter 改为换行，不再直接提交
- 新空闲会话标记（右上角蓝色小圆点 + 闪烁提醒），点击后消除
- 清除终端时同步清空后端 outputBuffer，修复清除后内容再次出现的问题
- xterm scrollback 限制 10000 行 + 后端 buffer 上限降至 2MB，防止长时间运行卡顿
- 终端实例按会话缓存，切换会话秒切无需 replay，大幅提升切换性能
- 补全 ElectronAPI 类型定义
