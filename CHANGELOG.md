# 更新日志

## [1.2.0] - 2026-03-26

### 会话管理与自动化增强

- 会话列表改为固定排序模式，支持拖拽调整顺序（基于 cwd 持久化，重启后保持）
- 默认主题改为 Ayu Dark（已有用户不受影响）
- 外部链接改用系统默认浏览器打开，不再在应用内弹窗确认
- 关闭应用时优雅退出 PTY 进程，修复 EXC_CRASH 崩溃
- 自动模式增加废话提问检测（中英文问句模式匹配），自动回复"继续，按你的方案执行"
- 新增 Rate Limit 自动恢复：检测到限流提示后解析重置时间，到时间自动发送 /resume（所有会话默认生效）
- Shift+Enter 改为换行，不再直接提交
- 新空闲会话标记（右上角蓝色小圆点 + 闪烁提醒），点击后消除
- 清除终端时同步清空后端 outputBuffer，修复清除后内容再次出现的问题
- xterm scrollback 限制 10000 行 + 后端 buffer 上限降至 2MB，防止长时间运行卡顿
- 终端实例按会话缓存，切换会话秒切无需 replay，大幅提升切换性能
- 补全 ElectronAPI 类型定义

## [1.1.0] - 2026-03-25

### 安全提示、用户引导与开发体验改进

- 启动时检测 Claude CLI 是否安装，未找到时弹出警告提示
- 新增首次运行欢迎弹窗，展示快捷键、功能说明和安全提示
- 安全提示说明权限绕过模式和 hooks 注入行为
- hooks 注入时增加控制台日志输出
- Shell 回退逻辑改进：优先使用 SHELL 环境变量，回退到 /bin/bash 而非 /bin/zsh
- package.json 新增 npm scripts：start、build:dmg、build:all、clean、typecheck
- 完善 README.md：平台支持、前置要求验证、快捷键表格、安全提示、故障排除、配置说明

## [1.0.0] - 2026-03-25

- 侧边栏面板高度持久化到 localStorage
- 大 buffer 分批写入 xterm，避免 UI 卡顿
- settings.json 写入前增加校验，防止损坏配置
- 历史项目列表过滤已不存在的路径
- 修复侧边栏拖拽事件监听器泄漏

- IPC 处理增加 session ID 校验
- 上次会话列表去重
